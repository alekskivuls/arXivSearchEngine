Detailed Description

InvertedIndex This data structure represents the inverted index. It incapsulates a set of stemmed terms and a list of Objects called DocInfos. The list of DocInfos represents an entire postings list. Each DocInfo within the postings list represents the positional occurences of the term that occurs for that document.
